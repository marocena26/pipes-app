Basic Pipe

Pipes included in Angular - Usually in the CommonModule

uppercase

Transforms text to all upper case.

{{ value_expression | uppercase }}
expression | uppercase {{ nameLower | uppercase }}

lowercase

Transforms text to all lower case.

{{ value_expression | lowercase }}
EXPRESSION | lowercase {{ nameUpper | lowercase }}

titlecase

Transforms text to title case. Capitalizes the first letter of each word and transforms the rest of the word to lower case. Words are delimited by any whitespace character, such as a space, tab, or line-feed character.

{{ value_expression | titlecase }}
eXPre sSIoN | titlecase {{ fullName | titlecase }}

date

Formats a date value according to locale rules.

  1. value_expression // {{ customDate }}
  2. value_expression | date // {{ customDate | date }}
  3. value_expression | date: 'short' // {{ customDate | date : "short" }}
  4. value_expression | date: 'long' // {{ customDate | date : "long" }}
  5. value_expression | date: 'MMMM' // {{ customDate | date : "MMMM" }}
  6. value_expression | date: 'MMMM dd, yyyy' // {{ customDate | date : "MMMM dd, yyyy" }}
  1. value_expression | date: 'long':'GMT-6' // {{ customDate | date : "long" : "GMT-6" }}
  2. value_expression | date: 'long':'GMT-4' // {{ customDate | date : "long" : "GMT-4" }}
  3. value_expression | date: 'long':'':'es' // {{ customDate | date : "long" : "" : "es" }}
  4. value_expression | date: 'long':'':'en-US' // {{ customDate | date : "long" : "" : "en-US" }}
  5. value_expression | date: 'long':'':'fr' // {{ customDate | date : "long" : "" : "fr" }}
