Uncommon Pipe

Pipes included in Angular - Usually in the CommonModule

i18nSelect

Generic selector that displays the string that matches the current value.

{{ value_expression | i18nSelect : mapping }}
Saludos {{ name }} , es un placer {{ gender | i18nSelect : invitationMap }} a nuestro evento.

Saludos {{ name }}, es un placer {{ gender | i18nSelect : invitationMap }} a nuestro evento.

i18nPlural

Maps a value to a string that pluralizes the value according to locale rules.

{{ value_expression | i18nPlural : pluralMap [ : locale ] }}
We currently {{ clients.length | i18nPlural : clientsMap }}

We currently {{ clients.length | i18nPlural : clientsMap }}

Slice

Creates a new Array or String containing a subset (slice) of the elements.

{{ value_expression | slice : start [ : end ] }}
Original
{{ value_expression }} // 
 {{ clients }}
slice:0:2
{{ value_expression | slice:0:2 }} // 
 {{ clients | slice : 0 : 2 }}
slice:1:2
{{ value_expression | slice:1:2 }} // 
 {{ clients | slice : 1 : 2 }}
slice:1:3
{{ value_expression | slice:1:3 }} // 
 {{ clients | slice : 1 : 3 }}
slice:3:4
{{ value_expression | slice:3:4 }} // 
 {{ clients | slice : 3 : 4 }}
slice:0:-3
{{ value_expression | slice:0:-3 }} // 
 {{ clients | slice : 0 : -3 }}

Json

Converts a value into its JSON-format representation. Useful for debugging.

{{ value_expression | json }} 
{{ person | json }}

KeyValue

Transforms Object or Map into an array of key value pairs.

{{ input_expression | keyvalue [ : compareFn ] }}
  • {{ item.key | titlecase }}: {{ item.value }} 
              
                <ul>
                  <li *ngFor="let item of person | keyvalue">
